> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix five UI issues in Structura: column width, window size, elide mode, metric tile root totals, and right-panel file-selected header redesign.

**Tech Stack:** Python 3.13, PySide6, unittest (offscreen via `QT_QPA_PLATFORM=offscreen`)

**Files:**
- Modify: `Structura.py:3576–3578` (metric tile update lines in `_refresh_view`)
- Test: `tests/test_scan_snapshot.py`

## Chunk 3: Right Panel File Header Redesign

### Task 3: Add compact file-selected header with styled info tiles

When a **file** (not a directory) is selected, the right panel currently uses the folder-style `_selection_title` label (28px bold) which overflows with long UUID filenames. Replace it with a compact header: a muted subtitle line (`📹 MOV · Feb 24, 2026`), the truncated filename, and three matching blue info tiles (TYPE / SIZE / FRAME RATE).

For **directory** selections the existing title/subtitle/metric-tiles layout is unchanged.

**Files:**
- Modify: `Structura.py` — `AnalyzerDashboard.__init__` (add new widget block)
- Modify: `Structura.py` — `AnalyzerDashboard._refresh_view` (show/hide logic)
- Test: `tests/test_scan_snapshot.py`

#### Step-by-step

def test_file_header_row_shown_for_file_selection_hidden_for_folder(self):

        scope = "folder" if stats.is_dir else "file"
        if not stats.is_dir:
            # File selected — show compact file header, hide folder title
            ext = Path(stats.path).suffix.lower()
            emoji = file_emoji(ext, is_dir=False)

            self._selection_title.show()

            self._selection_subtitle.setToolTip(stats.path)
